## 1. 索引器

索引器将原始区块链数据转换为可搜索、可查询的格式。它们为仪表板、分析、钱包、区块浏览器和应用程序后端提供支持。

### Envio

Envio 是区块链应用程序的数据层。它通过单个 GraphQL API 提供实时和历史链上数据，具有原始高速访问和 Envio Cloud 上的托管服务。

**能力**

- 实时和历史链上数据
- 具有原始高速访问的单个 GraphQL API
- 在 Envio Cloud 上托管

**文档**

Envio 的 HyperIndex 天生支持使用您自己的 RPC 作为数据源开箱即用地索引任何 EVM 链，包括 Stable。

[https://docs.envio.dev/](https://docs.envio.dev/)

### The Graph

为超过 75,000 个项目提供数据访问的去中心化索引协议。

**能力**

- 适用于 Stable 的 Subgraph
- 基于 GraphQL 的查询
- 分布式索引网络

**文档**

遵循此快速入门指南，在几分钟内创建、部署和查询 subgraph：

[https://thegraph.com/docs/en/developing/creating-a-subgraph/](https://thegraph.com/docs/en/developing/creating-a-subgraph/)

### Ormi Labs

Ormi 是一款下一代索引器，旨在大规模交付实时和历史区块链数据。

**能力**

- 链端索引
- 亚秒级查询延迟
- 无代码功能

**文档**

开始使用 Ormi 查询实时数据：

[https://docs.ormilabs.com/subgraphs/quickstart](https://docs.ormilabs.com/subgraphs/quickstart)

了解如何实时查询 Stable 上的 USDT0 数据：

[https://docs.ormilabs.com/subgraphs/tutorials/query-usdt0](https://docs.ormilabs.com/subgraphs/tutorials/query-usdt0)

### Goldsky

具有即时 subgraph 和开发人员工具的高性能索引平台。

**能力**

- Subgraph 部署和管理
- 用于实时事件流的 Webhook 创建
- 多 subgraph 同步到外部数据库

**文档**

[https://docs.goldsky.com/introduction](https://docs.goldsky.com/introduction)

如需 24/7 支持，请联系 [support@goldsky.com](mailto:support@goldsky.com)。

## 2. 分析提供商

分析工具可帮助团队跟踪网络活动、实际支付、仪表板、使用流程和合约交互。

### Allium

一个面向整个区块链生态系统工程师和分析师的基础数据平台。

**能力**

- 规范化的区块链数据集
- 分析团队的查询工具
- 企业数据基础设施

**文档**

开始使用 Allium 的数据平台和 API 资源：

[https://www.allium.so/](https://www.allium.so/)

**入门**：在 [allium.so](https://www.allium.so/) 注册以获取 API 访问权限，然后使用 REST API 查询规范化的 Stable 链上数据集。

### CoinMarketCap

全球最大的加密资产市场跟踪平台。

**能力**

- Stable 资产的价格跟踪
- 投资组合工具
- 市场数据 API

**文档**

探索 CMC API 和集成资源：

[https://coinmarketcap.com/api/](https://coinmarketcap.com/api/)

**入门**：在 [coinmarketcap.com/api](https://coinmarketcap.com/api/) 注册 API 密钥，并通过 REST API 查询 Stable 资产价格和市场数据。

### CoinGecko

全球最大的独立加密货币数据聚合器。

**能力**

- 市场列表和价格数据
- 历史分析
- 开发人员 API 访问

**文档**

访问 CoinGecko 的 API 文档：

[https://www.coingecko.com/en/api](https://www.coingecko.com/en/api)

**入门**：从 [coingecko.com/en/api](https://www.coingecko.com/en/api) 获取免费或专业 API 密钥，并通过 REST API 查询 Stable 代币价格和市场数据。

### Dexscreener

去中心化市场的实时图表和分析。

**能力**

- 实时 DEX 图表
- 流动性和交易对分析
- 交易仪表板

**文档**

探索 Dexscreener 的 API 端点和开发人员工具：

[https://docs.dexscreener.com/](https://docs.dexscreener.com/)

**入门**：浏览 [Dexscreener API 文档](https://docs.dexscreener.com/)，查询基于 Stable 的 DEX 池的实时交易对数据、流动性和交易活动。

### DeBank

以太坊和 EVM 生态系统的投资组合跟踪器。

**能力**

- 钱包分析
- 交易摘要
- 跨链投资组合跟踪

**文档**

阅读 DeBank 的 API 参考和集成文档：

[https://docs.debank.com/](https://docs.debank.com/)

**入门**：注册 [DeBank Cloud](https://cloud.debank.com/) 以获取 API 密钥，并查询 Stable 钱包余额、交易历史和投资组合数据。

## 3. 合规和调查

合规和调查平台帮助团队监控交易风险并追踪链上资金流。

### Chainalysis

Chainalysis 通过其 [KYT](https://www.chainalysis.com/product/kyt/)、实体筛选产品和 [Reactor](https://www.chainalysis.com/product/reactor/) 支持 Stable。

当部署在 Stable 上的新的可替代和不可替代代币遵循主要标准（如 ERC-20 和 ERC-721）时，它们将自动获得覆盖。

**能力**

- 通过 KYT 对 Stable 代币进行持续监控和可操作警报
- 对 Stable 进行实体筛选
- 跨 Stable 代币的资金流追踪和交易调查
- 自动覆盖遵循 ERC-20 和 ERC-721 等标准的新代币

**文档**

阅读 [Stable 集成公告](https://www.chainalysis.com/blog/stable-automatic-token-support/)以获取覆盖详细信息。

[申请 Chainalysis 演示](https://www.chainalysis.com/request-a-demo/)。
