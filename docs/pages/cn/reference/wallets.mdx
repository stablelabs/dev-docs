## 类别指南

- **用户钱包：** 这些是传统的面向消费者的钱包，例如移动应用程序、浏览器扩展或交易所关联钱包。它们允许用户持有 USDT、进行转账、连接到 dApp 并直接与 Stable 交互。
- **钱包 SDK：** 一种软件开发工具包，为开发人员提供预构建的工具、API 和基础设施，以便将其应用程序直接集成钱包创建、密钥管理、交易签名和区块链交互。
- **托管 / 机构钱包：** 为机构提供受监管的、企业级资产托管的平台。这些解决方案侧重于合规性、治理控制、安全密钥管理和资金操作，而不是终端用户流程。
- **嵌入式 / 应用内钱包：** 通过 SDK 或后端系统在应用程序内生成的钱包。这些钱包可以为主流用户提供无缝的入职体验，而无需他们安装或理解外部加密钱包。
- **智能钱包 / 账户抽象：** 支持自定义逻辑（例如无 gas 交易、捆绑操作或自动化执行）的可编程钱包。这些钱包通过开发人员定义的行为扩展了基本的钱包功能。
- **MPC 钱包提供商：** 使用多方计算 (MPC) 来将私钥控制分发到多个参与方或设备之间的密钥管理系统。非常适合需要高安全性托管而无需传统助记词的应用程序或企业。
- **连接提供商：** WalletConnect (Reown) 等协议，用于连接钱包和 dApp。它们不存储资产，也不充当钱包本身；相反，它们提供安全的通信渠道，用于交易签名和交互。

## 1. 用户钱包

这些是全球主要交易所提供的终端用户钱包。它们允许用户持有 USDT、转移资金并连接到 Stable 上的应用程序。

### Stable Pay

Stable 上构建的非托管支付钱包，专为快速、稳定币原生交易而设计。Stable Pay 提供即时 USDT 支付、可预测的费用以及针对日常转账和商业优化的简单用户体验。

**功能**

- Stable 的非托管钱包
- 即时 USDT 支付
- 可预测且一致的交易成本
- 直接构建在 Stable 的 USDT 原生结算层上
- 为支付和商业设计的消费者友好型 UI

### 币安钱包

全球交易量最大的交易所集成使用的多链钱包。

**功能**

- 支持 Stable USDT
- 与币安生态系统直接集成
- 移动端和扩展程序钱包选项

### Bitget 钱包

连接到 Bitget 交易所生态系统的多资产钱包，支持加密货币、股票和 ETF。

**功能**

- 支持 Stable USDT
- 内置 dApp 浏览器
- 与 Bitget 交易账户无缝集成

### Gate 钱包 (Gate Onchain)

由全球最大的现货交易所之一支持的钱包产品。

**功能**

- 支持 Stable USDT
- Gate 交易所和钱包之间轻松转账
- dApp 和网络应用程序连接

### OKX 钱包 (OKX Onchain)

全球使用的强大多链钱包。

**功能**

- 支持 Stable USDT
- 深度 OKX 生态系统集成
- Web 端、移动端和扩展程序钱包选项

### Atomic Wallet

可在桌面、移动和作为浏览器扩展程序使用的非托管多资产钱包。Atomic Wallet 集成了 [Simplex by Nuvei](/cn/reference/ramps#simplex-by-nuvei) 入口，因此用户可以通过内置的 Simplex 在应用程序内通过银行卡、Apple Pay、Google Pay 或银行转账直接在 Stable 上购买 USDT。

**功能**

- 在 Stable 上持有、发送和接收 USDT
- 私钥本地存储在用户设备上
- 通过 Simplex 在应用程序内购买 Stable USDT
- 可用于 Windows、macOS、Linux、iOS、Android 和 Chrome

**开始使用**：从 [atomicwallet.io](https://atomicwallet.io/downloads) 下载 Atomic Wallet，将 Stable 添加为网络，然后使用应用程序内购买流程通过 Simplex 购买 Stable USDT。

## 2. 钱包 SDK

### Tether 钱包开发套件 (WDK)

Tether 提供的开源 SDK，用于在任何平台和区块链上构建自托管钱包。

**功能**

- 多链支持：比特币、以太坊、TON、TRON、Solana、Spark 等
- 代理钱包：原生支持 AI 代理钱包和 Stable 上的 x402 支付
- DeFi 集成：支持互换、桥和借贷协议的插件
- 可扩展设计：添加自定义模块以支持新区块链或协议

**开始使用**：安装 [`@tetherto/wdk`](https://www.npmjs.com/package/@tetherto/wdk) 和 [`@tetherto/wdk-wallet-evm`](https://www.npmjs.com/package/@tetherto/wdk-wallet-evm)，然后按照 [WDK 文档](https://docs.wallet.tether.io) 配置 Stable 作为您的目标链。

## 3. 托管和机构钱包

### Anchorage

一家联邦特许的国家银行，为数字资产提供机构级托管服务。

**功能**

- 安全托管 Stable USDT
- 完全合规和监管监督
- 企业级密钥管理和访问控制

## 4. 嵌入式 / 应用内钱包

通过 SDK 直接嵌入到应用程序中的钱包，实现无缝的用户 onboarding 和支付流程。

### Coinbase Developer Platform (CDP)

CDP 为面向用户的应用程序和后端系统提供非托管钱包基础设施。用户认证钱包使用电子邮件、短信或社交登录。API 密钥钱包允许服务器管理账户，而 CDP 在受信任的执行环境中保护私钥。

**功能**

- 可通过电子邮件、短信或社交登录且无需助记词的嵌入式钱包
- 用于自动化工作流、代理和后端服务的 API 密钥钱包
- 钱包创建、交易签名、余额、转账、互换、质押以及法定资产入口和出口
- 支持 EVM 兼容网络（包括 Stable）和 Solana
- 用户控制钱包的密钥导出

**开始使用**：按照 [CDP 非托管钱包文档](https://docs.cdp.coinbase.com/wallets/non-custodial-wallets/overview)为您的应用程序选择用户认证或 API 密钥快速入门。

### Dynamic

企业级钱包基础设施，服务于数千个应用程序和超过 4000 万用户。

**功能**

- 钱包创建和认证
- 嵌入式钱包流
- 应用程序和金融科技的用户 onboarding

**开始使用**：按照 [Dynamic SDK 设置文档](https://docs.dynamic.xyz/introduction/welcome) 安装 SDK 并在您的应用程序中配置 Stable 作为支持的网络。

### Reown (前身为 WalletConnect)

一种广泛采用的连接钱包和应用程序的标准。

**功能**

- 安全的钱包到 dApp 连接
- 支持移动、桌面和扩展钱包
- 广泛的生态系统兼容性

**Reown SDK 用于钱包 onboarding**

Stable 支持与 **Reown SDK** 集成，以帮助开发者为用户提供无缝的钱包和 onboarding 体验。

Reown 提供了一个开源的一体化 SDK，作为 WalletConnect 网络的官方网关。它可以在您的应用程序中实现流畅的钱包连接、交易、登录、嵌入式钱包（电子邮件和社交登录）、链上支付、应用内互换等。

**开始使用**

- 访问 Reown 的文档：[https://docs.reown.com/overview](https://docs.reown.com/overview)

## Stable 网络设置

### 将 Stable 添加到您的钱包

**网络参数：**

- **网络名称：** Stable
- **链 ID：** 988
- **货币：** USDT0
- **RPC URL：** https://rpc.stable.xyz
- **区块浏览器：** https://stablescan.xyz

### 构建钱包集成

您可以通过以下方式添加 Stable 支持：

- 通过 Stable RPC 启用签名和 Gas 估算
- 支持 USDT 原生转账
- 集成 WalletConnect 以连接 dApps
- 将 Stable 添加到链列表或元数据注册表
