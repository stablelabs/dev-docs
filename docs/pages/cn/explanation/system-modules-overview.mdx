# 概览

Stable 通过**系统模块**暴露核心结算行为，以**预编译合约**的形式实现，确保gas效率和可预测的控制。

**主要模块：**

- [银行模块](./bank)
  - 处理 USDT 转账、余额记账和托管流程
- [分发模块](./distribution)
  - 网络参与者的费用分发和奖励逻辑
- [质押模块](./staking)
  - 控制验证者参与和质押（将在主网推出）

**DApps 可以利用内置模块，而不需要重新实现代币或结算逻辑。**
