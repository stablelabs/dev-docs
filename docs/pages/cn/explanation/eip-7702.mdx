# EIP-7702

Stable 支持 EIP-7702，引入了统一账户模型，使外部账户（EOA）可临时以智能账户方式运行。

关键特性：

- 用户可继续使用原有的私钥签名交易
- 执行时可临时具备智能合约逻辑，无需永久升级账户
- 实现安全的支付与托管授权流程
- 改善钱包与商户使用 USDT 时的体验

开发者：

- 用户无需部署新合约即可获得合约级功能，降低接入门槛，简化托管逻辑。
