# Stable의 계정

Stable의 계정은 [EIP-7702 위임](/en/explanation/eip-7702)을 통해 선택적으로 스마트 컨트랙트 로직을 실행할 수 있는 표준 Ethereum EOA입니다. 사용자는 지갑, 일괄 결제, 정기 구독, 세션 키 전반에서 하나의 주소와 하나의 개인 키를 유지합니다. 에이전트는 어떠한 커스터디 미들웨어 없이도 동일한 계정 모델을 실행합니다.

## 구축할 수 있는 것

- 시드 문구로부터 만드는 **지갑**, 네이티브 USDT0 잔액 조회 및 서명된 트랜잭션 포함.
- **일괄 결제**: 위임된 EOA를 통해 여러 전송을 하나의 원자적 트랜잭션으로 실행.
- **지출 한도**: 위임 로직을 통해 EOA 자체에 트랜잭션당 또는 일일 상한을 적용.
- **세션 키**: 사용자가 모든 작업마다 다시 서명하지 않도록 dApp에 범위, 시간, 예산이 제한된 키를 부여.
- **에이전트 지갑**: AI 에이전트에 자체 커스터디 키로 자금을 지원하고 x402 서비스 비용을 자율적으로 지불하게 함. 제공업체 및 통합 패턴은 [에이전트 지갑](/en/reference/agentic-wallets)을 참조하세요.

## Stable의 차별점

- **모든 것을 위한 하나의 주소.** 스마트 컨트랙트 기능을 사용하기 위한 계정 마이그레이션이 필요 없습니다. EIP-7702는 기존 EOA *위에* 코드를 위임합니다.
- **USDT0 전용 가스.** 사용자에게 별도의 네이티브 토큰이 필요 없습니다. 새 계정은 USDT0로 자금을 충전하고 즉시 트랜잭션을 실행할 수 있습니다.
- **다기능 위임 패턴.** 단일 위임으로 일괄 처리, 지출 한도, 세션 키, 구독을 결합할 수 있어 하나의 위임이 출시하는 모든 기능을 포괄합니다.
