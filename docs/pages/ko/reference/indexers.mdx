# 인덱서

인덱서, 분석 플랫폼 및 블록체인 인텔리전스 제공업체는 Stable에서 온체인 데이터를 쿼리하고, 활동을 측정하고, 거래 위험을 모니터링하고, 자금 흐름을 조사하는 데 도움을 줍니다. Stable은 EVM 호환이므로 표준 이더리움 인덱싱 도구가 원활하게 작동합니다.

이 페이지에는 Stable에서 인덱싱, 분석, 규정 준수 및 조사를 위해 사용할 수 있는 제공업체가 나열되어 있습니다.

## 1. 인덱서

인덱서는 원시 블록체인 데이터를 검색 가능하고 쿼리 가능한 형식으로 변환합니다. 대시보드, 분석, 지갑, 블록 탐색기 및 애플리케이션 백엔드를 지원합니다.

### Envio

Envio는 블록체인 애플리케이션을 위한 데이터 레이어입니다. 단일 GraphQL API를 통해 실시간 및 과거 온체인 데이터를 제공하며, Envio Cloud에서 원시 고속 액세스 및 관리형 호스팅을 제공합니다.

**기능**

- 실시간 및 과거 온체인 데이터
- 원시 고속 액세스를 제공하는 단일 GraphQL API
- Envio Cloud에서 관리형 호스팅

**문서**

Envio의 HyperIndex는 Stable을 포함한 모든 EVM 체인에 대한 인덱싱을 데이터 소스로 자체 RPC를 사용하여 기본적으로 지원합니다.

[https://docs.envio.dev/](https://docs.envio.dev/)

### The Graph

75,000개 이상의 프로젝트에 데이터 액세스를 제공하는 분산형 인덱싱 프로토콜입니다.

**기능**

- Stable용 서브그래프
- GraphQL 기반 쿼리
- 분산 인덱싱 네트워크

**문서**

이 빠른 시작 가이드를 따라 몇 분 안에 서브그래프를 생성, 배포 및 쿼리하세요.

[https://thegraph.com/docs/en/developing/creating-a-subgraph/](https://thegraph.com/docs/en/developing/creating-a-subgraph/)

### Ormi Labs

Ormi는 실시간 및 과거 블록체인 데이터를 대규모로 제공하기 위해 구축된 차세대 인덱서입니다.

**기능**

- 체인 인덱싱의 팁
- 1초 미만의 쿼리 지연 시간
- 노코드 기능

**문서**

Ormi로 실시간 데이터 쿼리를 시작하세요.

[https://docs.ormilabs.com/subgraphs/quickstart](https://docs.ormilabs.com/subgraphs/quickstart)

Stable에서 USDT0 데이터를 실시간으로 쿼리하는 방법을 알아보세요.

[https://docs.ormilabs.com/subgraphs/tutorials/query-usdt0](https://docs.ormilabs.com/subgraphs/tutorials/query-usdt0)

### Goldsky

인스턴트 서브그래프 및 개발 도구를 갖춘 고성능 인덱싱 플랫폼입니다.

**기능**

- 서브그래프 배포 및 관리
- 실시간 이벤트 스트리밍을 위한 웹훅 생성
- 외부 데이터베이스로의 다중 서브그래프 동기화

**문서**

[https://docs.goldsky.com/introduction](https://docs.goldsky.com/introduction)

연중무휴 24시간 지원을 받으려면 [support@goldsky.com](mailto:support@goldsky.com)으로 문의하세요.

## 2. 분석 제공업체

분석 도구는 팀이 네트워크 활동, 실제 결제, 대시보드, 사용 흐름 및 계약 상호 작용을 추적하는 데 도움이 됩니다.

### Allium

블록체인 생태계 전반의 엔지니어와 분석가를 위한 기본 데이터 플랫폼입니다.

**기능**

- 정규화된 블록체인 데이터셋
- 분석 팀을 위한 쿼리 도구
- 엔터프라이즈 데이터 인프라

**문서**

Allium의 데이터 플랫폼 및 API 리소스 시작하기:

[https://www.allium.so/](https://www.allium.so/)

**시작하기**: [allium.so](https://www.allium.so/)에서 가입하여 API 액세스를 얻은 다음 REST API를 사용하여 정규화된 Stable 온체인 데이터셋을 쿼리합니다.

### CoinMarketCap

암호화 자산을 위한 세계에서 가장 큰 글로벌 시장 추적 플랫폼입니다.

**기능**

- Stable 자산 가격 추적
- 포트폴리오 도구
- 시장 데이터 API

**문서**

CMC API 및 통합 리소스 탐색:

[https://coinmarketcap.com/api/](https://coinmarketcap.com/api/)

**시작하기**: [coinmarketcap.com/api](https://coinmarketcap.com/api/)에서 API 키를 등록하고 REST API를 통해 Stable 자산 가격 및 시장 데이터를 쿼리합니다.

### CoinGecko

세계에서 가장 큰 독립 암호화폐 데이터 수집기.

**기능**

- 시장 목록 및 가격 데이터
- 과거 분석
- 개발자용 API 액세스

**문서**

CoinGecko의 API 문서 액세스:

[https://www.coingecko.com/en/api](https://www.coingecko.com/en/api)

**시작하기**: [coingecko.com/en/api](https://www.coingecko.com/en/api)에서 무료 또는 Pro API 키를 얻고 REST API를 통해 Stable 토큰 가격 및 시장 데이터를 쿼리합니다.

### Dexscreener

분산형 거래소를 위한 실시간 차트 및 분석.

**기능**

- 라이브 DEX 차트
- 유동성 및 페어 분석
- 거래 대시보드

**문서**

Dexscreener의 API 엔드포인트 및 개발 도구 탐색:

[https://docs.dexscreener.com/](https://docs.dexscreener.com/)

**시작하기**: [Dexscreener API 문서](https://docs.dexscreener.com/)를 탐색하여 Stable 기반 DEX 풀에 대한 실시간 페어 데이터, 유동성 및 거래 활동을 쿼리합니다.

### DeBank

이더리움 및 EVM 생태계를 위한 포트폴리오 추적기.

**기능**

- 지갑 분석
- 거래 요약
- 체인 전체 포트폴리오 추적

**문서**

DeBank의 API 참조 및 통합 문서 읽기:

[https://docs.debank.com/](https://docs.debank.com/)

**시작하기**: [DeBank Cloud](https://cloud.debank.com/)에 가입하여 API 키에 액세스하고 Stable 지갑 잔액, 거래 내역 및 포트폴리오 데이터를 쿼리합니다.

## 3. 규정 준수 및 조사

규정 준수 및 조사 플랫폼은 팀이 거래 위험을 모니터링하고 온체인 자금 흐름을 추적하는 데 도움이 됩니다.

### Chainalysis

Chainalysis는 KYT, 엔터티 스크리닝 제품 및 Reactor를 통해 Stable을 지원합니다.

[Stable 통합 발표](https://www.chainalysis.com/blog/stable-automatic-token-support/)는 새로운 토큰에 대한 자동 적용을 확인합니다. 여기에는 ERC-20 및 ERC-721과 같은 주요 표준을 따르는 토큰이 포함됩니다.

**기능**

- KYT를 통한 Stable 토큰에 대한 지속적인 모니터링 및 실행 가능한 경고
- Stable에 대한 엔터티 스크리닝
- Stable 토큰 전반의 자금 흐름 추적 및 거래 조사
- ERC-20 및 ERC-721과 같은 표준을 따르는 새로운 토큰에 대한 자동 적용

**문서**

적용 범위 세부 정보는 [Stable 통합 발표](https://www.chainalysis.com/blog/stable-automatic-token-support/)를 읽어보세요.

[Chainalysis 데모 요청](https://www.chainalysis.com/request-a-demo/).
